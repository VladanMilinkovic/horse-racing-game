# Horse Racing Game

Interactive horse racing game built with Vue 3 and Vuex.

## Requirements

- Node.js 22+
- npm

## Setup

```bash
npm install
```

## Run locally

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tests

### Unit tests

```bash
npm run test:unit
```

### E2E tests (Playwright)

Install Playwright browser binaries once:

```bash
npx playwright install chromium
```

Then run:

```bash
npm run test:e2e
```
